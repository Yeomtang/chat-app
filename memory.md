# 프로젝트 메모리

## 프로젝트 개요
방송 제작 배경 LED 화면용 실시간 익명 채팅 시스템

## 핵심 요구사항
- **용도**: 방송 배경 LED 화면에 상시 노출되는 채팅창
- **참여자**: 현장 토론자 + 관객 (익명 채팅)
- **입장 방식**: QR코드 스캔 → 웹페이지 입장 (앱 다운로드 없음)
- **네트워크**: LTE 환경에서도 원활하게 동작해야 함 (Wi-Fi 의존 X)
- **채팅 UI**: iMessage 스타일, 메시지가 위로 계속 올라오는 구조
- **익명성**: 관객은 익명으로 참여

## 화면 구성
- 비율: 2:3 세로형 (핸드폰 세로 비율 기준, LED 특성 반영)
- 화면 수: 1개
- 테마: 라이트모드
- 채팅 스타일: 깔끔한 말풍선, 좌측 정렬 (첨부 이미지 참고)
- 입장 시 닉네임: **직장인 피로 테마 랜덤 닉네임 자동 추천**
  - 예: "야근 3년차", "퇴근하고 싶다", "월요일이 싫은 사람" 등
  - 닉네임은 LED 화면 말풍선에도 표시됨

## 화면 모드 (2가지)
1. **채팅 모드 (기본)**: 채팅창이 전체 화면 차지, iMessage 스타일로 위로 흐름
2. **질문 모드 (제작진 트리거)**: 채팅창 블러 처리 → 공통 질문 팝업 등장
   - 예시 질문: "지금 본인의 회사 생활에 만족하십니까?"
   - 답변 형식 2가지:
     - **객관식 (Yes/No)**: 실시간 비율 시각화
       - YES 원 / NO 원 두 개의 원형 클러스터
       - 투표마다 작은 원(공)이 해당 원으로 날아들어오는 애니메이션
       - 공들이 반대쪽 원으로 이동하는 흐름으로 실시간 비율 변화 표현
       - 답변 변경 가능, 단 **30초 타이머** 제한
       - 30초 카운트다운: **LED 화면 + 관객 핸드폰 양쪽에 표시**
       - 타이머 종료 후 최종 결과 고정
       - 흐름: 질문 등장 → 30초 카운트다운 → 실시간 공 이동 → 종료 → 결과 확정
     - **주관식 (자유 텍스트)**: 개별 답변 흘려보내기 or 워드클라우드

## 관리자 인터페이스 (제작진용)
- 질문을 **미리 여러 개 작성**해두고 순서대로 실행
- 기능: 질문 입력/편집 → 순서 정렬 → 시작 버튼 → 결과 확인
- 방송 중 즉흥적으로 질문 추가도 가능해야 할 듯

## 전체 화면 흐름
1. **채팅 모드**: iMessage 스타일 익명 채팅, 위로 스크롤
2. **질문 모드 진입** (제작진 트리거): 채팅 블러 → 질문 텍스트 + 30초 타이머 등장
3. **투표 중**: YES/NO 공 애니메이션 실시간, 타이머 LED+핸드폰 동시 표시
4. **타이머 종료**: 질문 텍스트·타이머 사라짐 → **결과(원형 클러스터)만 화면에 floating**
5. **결과 유지**: MC/토론자 코멘트 시간, 제작진이 수동으로 다음 전환
6. **채팅 모드 복귀** (제작진 수동 트리거): 다시 익명 채팅, 관객이 토론자에게 질문/의견 전송

## 미결 사항 / 논의 중
- ✅ 질문/투표 모드 중 채팅 입력 완전 차단 → 오직 답변에만 집중
- ✅ 투표 후 핸드폰 화면: 내가 선택한 쪽 하이라이트 + 실시간 비율 표시
- 채팅 모드 ↔ 질문 모드 전환 시 LED 화면 애니메이션
- 주관식 결과 표시 방식 세부 (흘려보내기 vs 워드클라우드 선택)
